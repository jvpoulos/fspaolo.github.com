Poster at AGU 2011, San Francisco

Long term elevation changes on Antarctic ice shelves from multi-mission RA.

This poster was made using LaTeX and the Beamerposter package. I’ve created a custom theme beamerthemeAGU2011.sty based on one of the default themes. If you are interested in using (and modifying) this template, download the tarball with the LaTeX source code and figures (PDF of the poster included): paolo_agu2011.tgz

Decompress the tarball:

$ tar xzf paolo_agu2011.tgz

Compile the source code to generate a PDF:

$ cd paolo_agu2011
$ pdflatex paolo_agu2011.tex
